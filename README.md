#  YT Cards UI Project

A responsive YouTube-style card layout built using HTML, CSS Grid, and Bootstrap carousel.
The project focuses on clean UI design, proper layout structure, and full mobile responsiveness.

##  Features
- Responsive card layout using CSS Grid
- Bootstrap image carousel
- Mobile-friendly design
- Clean UI with gradients
- External YouTube links

##  Technologies Used
- HTML5
- CSS3
- Bootstrap 5

##  Responsive Design
- Desktop: 3 cards per row
- Tablet: 2 cards per row
- Mobile: 1 card per row
